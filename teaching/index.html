    1. Big Data Analytics On Azure Data Platform

    Oct 2021 | Sample Teaching Session delivered face to face at an academic institute

    Slides

    This talk presents an architectural overview of migrating big data analytics workload to azure cloud data platform. I discuss the key aspects and capabilities of : 1) Azure Data factory as an orchestrator and ETL tool 2) Ingestion in Azure Data Warehouse and 3) Storing current and historical files in Azure Data Lake
